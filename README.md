<h4 align='center'>Pega as cotações do dólar, euro, libra esterlina e do ouro</h4>

## ⬇️ Como instalar (Precisa do python)

```bash
git clone https://github.com/kauefraga/AutoQuotation.git
cd AutoQuotation
pip install -r ./requirements.txt
python src/main.py
```
